# TextSay - Text to Speech Converter

## Description
TextSay is a simple web-based application that converts text input into speech using the browser's built-in Speech Synthesis API. Users can enter text in a textarea and click a button to hear the text spoken aloud.

## Features
- Clean, responsive user interface with a gradient background
- Real-time text-to-speech conversion
- Error handling for empty input or speech synthesis issues
- Play/pause functionality for speech output
- Mobile-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Web Speech API

## How to Use
1. Open `index.html` in a modern web browser that supports the Web Speech API (e.g., Chrome, Firefox, Safari)
2. Enter your desired text in the textarea
3. Click the "Play Converted Sound" button to start speech synthesis
4. The button will change to "Sound is Playing..." while audio is playing
5. Click again to stop the speech if needed

## Browser Support
This application uses the Web Speech API, which is supported in:
- Google Chrome (recommended)
- Mozilla Firefox
- Microsoft Edge
- Safari

## File Structure
- `index.html` - Main HTML file
- `style.css` - CSS styles for the application
- `script.js` - JavaScript logic for text-to-speech functionality
